Detailed Description

template<typename T>
class Muscle3Grid< T >

Helper class for accessing grids/matrices received via MUSCLE3

This makes it easier to access individual elements of a grid/matrix. I'm going to add that feature to the libmuscle API and then this won't be needed any more, but until then here we are.

The object, once created, references the data object that was passed in, so you need to make sure that that still exists for as long as you use the Muscle3Grid object.
